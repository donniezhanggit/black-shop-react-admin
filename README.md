# black-shop-vue-admin

黑店后台管理系统，基于 antd-design-pro 开发采用 react + redux + dva + umi + typescript 等开源技术进行开发

## 安装

```bash
npm install
```

或者

```bash
yarn
```

### 启动项目

```bash
npm start
```

或者

```bash
yarn start
```

### 打包

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
